# drift

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

Lint/format with OXC:

```bash
bun run lint
bun run lint:fix
bun run lint:ast-grep
bun run format
bun run format:check
bun run check
```

This project was created using `bun init` in bun v1.3.4. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
